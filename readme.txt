Offline payment that allows you to accept Gcash payments by scanning your QR Code.

== Frequently Asked Questions ==

= How does it work? =

The plugin allows you to upload your Gcash QR code which can then be scanned by your customers upon checkout. They will be required to enter the payment reference ID before completing the order. You will then need to manually verify each payment and update your order statuses accordingly.

= How can I contribute =

The plugin is open source and contributions are welcome: https://github.com/jpneey/WooCommerce-Gcash-Payment
